# Secant

Secant is a modern audio workstation (MIDI sequencer) designed for rapid song development and ease of use. It features a unique chord-based editor that allows you to input chords and modifications alongside traditional plain notes, streamlining the composition process.

## Features

*   **Chord-Based Editing**: Input chords and apply modifications directly, bridging the gap between music theory and sequencing.
*   **Hybrid Workflow**: Combine chord structures with individual note editing for maximum flexibility.
*   **Rapid Prototyping**: Designed to help you get your ideas down quickly without getting bogged down in technical details.
*   **User-Centric Interface**: Focused on usability and an intuitive experience.
